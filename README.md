# Sentiment Analysis using TextBlob (Dockerized)

This project performs sentiment analysis on text data using TextBlob, a Python NLP library.
Each input sentence is analyzed and classified as Positive, Negative, or Neutral based on polarity.

The project is containerized using Docker, allowing it to run with a single command on any system.
